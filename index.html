Now we are going to discuss about a small topic which is HTML tags

In HTML language there are three types of tags First one is called opening tag,closing tag and another one is called self-closing tag.

Opening Tag

<html> <head> these are the examples of opening tags

Closing Tags

</html> </head> these are the examples of closing tags

Self-Closing Tags

<hr> <br> <img> <input> these are the examples of self-closing tags

combination of opening tag and losing tag is called Element.
